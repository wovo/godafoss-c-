xy

template< typename T = xy_value_type, T zero = 0 >
struct xy final {
   using value_t = xy_value_type;
   value_t x, y;  ... };

The xy< T > ADT class template is a pair of two T values named x and y. It is used for distances in an xy plane, like on a window or terminal.

The xy< torsor< T > > is used to identify a location on an xy plane.
