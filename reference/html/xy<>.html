xy<>

template< typename T = xy_value_type, T zero = 0 >
struct xy final {
   using value_t = xy_value_type;
   value_t x, y;  ... };

The xy< T > ADT class template is a pair of two T values named x and y. It is used for distances in an xy plane, like on a window or terminal. The xy< torsor< T > > is used to identify a location on an xy plane. 

   constexpr xy( value_t x, value_t y ): x{ x }, y{ y }{}
   constexpr xy():x{ zero }, y{ zero }{}  ... };

The default constructor intializes a and y to the zero value.
